### ความเป็นมา

App นี้เกิดขึ้นจากประสบการณ์จริงของผู้พัฒนา

ผมเคยเปิดบริษัทรับออกแบบและสร้างบ้านน็อคดาวน์ด้วยความฝันที่อยากให้คนทั่วไปมีบ้านที่ดีในราคาที่เอื้อมถึงได้ แต่เมื่อวิกฤต COVID-19 ปี 2019 มาถึง กิจการต้องปิดตัวลง เหมือนกับธุรกิจอีกนับพันแห่งทั่วโลก

หลังจากตกงานและหยุดนิ่งอยู่กับตัวเองสักพัก ผมตัดสินใจเรียนรู้สิ่งใหม่ นั่นคือการพัฒนา App ด้วยตัวเอง

**Inno Home Smart Home** คือ App ที่ 2 ที่ผมสร้างขึ้น — มันไม่ได้สมบูรณ์แบบ แต่มันคือก้าวที่ภูมิใจที่สุดก้าวหนึ่งในชีวิต

## 🇬🇧 English

### Background

This app was born from the developer's real-life experience.

I once ran a company designing and building prefabricated modular homes — driven by a dream to make quality housing accessible to everyone. Then COVID-19 arrived in 2019, and like thousands of businesses around the world, my company had to close its doors.

After losing my job and spending time reflecting, I made a decision: to learn something new and build something with my own hands. That something was app development.

**Inno Home Smart Home** is the second app I have ever built. It may not be perfect — but it is one of the proudest steps I have ever taken.

### Vision

> Affordable homes can be smart homes too — technology should not be a privilege reserved for the wealthy.

**Inno Home** is designed for two core groups of users:

**1. The Elderly**
Voice commands in Thai — no buttons to remember, no walking across the room. Just speak, and it works.

**2. Budget Homeowners**
A Smart Home system that doesn't require a massive investment. Use the old smartphone you already own as the controller.

### Features (v2.8)

| Feature | Description |
|---|---|
| 🏠 **Home Screen** | Real house photo + interactive floor plan |
| 🎬 **Scene Presets** | Home / Sleep / Away / Movie / Morning |
| 💡 **Device Control** | Dimmable lights / adjustable AC / all appliances |
| 🎙️ **Voice Command** | Speak Thai naturally via Web Speech API |
| 📊 **Activity Log** | Auto-sync to Google Sheets |
| 📱 **Mobile First** | Full-screen Android & iOS support |

### Appliances Supported

```
Bedroom      →  Air Conditioner, Light, TV
Living Room  →  TV, Light
Kitchen      →  Fridge, Microwave, Oven, Coffee Maker, Electric Stove, Light
Bathroom     →  Water Heater, Light
Outdoor      →  Front Porch, Back, Side Lights, Fence Lights
```

### Tech Stack

```
Frontend    Pure HTML + CSS + JavaScript (no framework)
Voice       Web Speech API (free, Thai language support)
AI Vision   Google Gemini 2.5 Flash API
Database    Google Sheets + Apps Script
Hosting     GitHub Pages
```

### How to Use

**On Mobile (Recommended)**
1. Open the link in Chrome (Android) or Safari (iOS)
2. Tap **Add to Home Screen** to use it like a native app
3. Tap 🎙️ and speak your command in Thai

**Example Voice Commands**
```
"เปิดไฟทั้งหมด"  →  Turn on all lights
"ปิดแอร์"        →  Turn off the AC
"โหมดนอนหลับ"   →  Activate sleep mode
"ชงกาแฟ"         →  Start the coffee maker
"ปิดทุกอย่าง"   →  Turn everything off
```
